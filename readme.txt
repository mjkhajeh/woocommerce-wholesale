Changes the price of the product in the shopping cart according to its quantity

== Description ==
You can specify a minimum quantity for a product. When the user adds the product to the cart and if the number of wanted quantity was greater than the minimum you specific, the price of the product will change to what you want.
That price can be a percent or a specific value.
This is wholesale.

= One more thing =
The plugin using i18n, So you can translate it for your language.
The plugin has been translated to Persian already.

[GitHub Repo](https://github.com/mjkhajeh/woocommerce-wholesale)

== Installation ==

1. Upload the the plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. After activating, in the general tab of product metabox under the price section, wholesale fields will appear.
4. Enjoy!
